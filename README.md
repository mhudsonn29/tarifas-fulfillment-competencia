# 📦 Tarifas Fulfillment - Competencia Chile

> Scraper automático que obtiene las tarifas de fulfillment de Falabella, Mercado Libre y Paris cada 48 horas.

## 🌐 Ver Tarifas

**👉 [Ver tarifas actualizadas](https://mhudsonn29.github.io/tarifas-fulfillment-competencia/)**

## ⚙️ Cómo Funciona

1. **GitHub Actions** ejecuta el scraper automáticamente cada 48 horas
2. Los datos se extraen de las páginas públicas de cada marketplace
3. Se genera una página web estática con las tarifas
4. La página se publica en GitHub Pages

## 📊 Marketplaces Incluidos

### Falabella (Fulfillment by Falabella)
- Tarifa de Retiro / Tren logístico
- Servicio de etiquetado
- Almacenamiento diario
- Almacenamiento prolongado
- Cofinanciamiento logístico
- Retiro anticipado

### Mercado Libre (Full)
- Costos por colecta
- Cargos por incumplimiento
- Almacenamiento
- Stock antiguo
- Retiro o descarte

### Paris (Fulfillment by Paris)
- Costos de despacho a domicilio
- Almacenamiento
- Retiro

## 🚀 Ejecutar Manualmente

Puedes forzar una actualización yendo a **Actions** > **Scrape Tarifas** > **Run workflow**
